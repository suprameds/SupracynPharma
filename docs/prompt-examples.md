# Prompt Examples

## Architecture
Use `next-architect` to design a scalable app router structure for a pharmacy admin panel with orders, prescriptions, and users.

## Database
Use `db-agent` to design Supabase schema for customers, orders, prescriptions, and payments with indexes and RLS notes.

## Backend
Use `api-backend-agent` to create a secure POST route for creating an order with validation and predictable error responses.

## UI
Use `ui-builder` to create a responsive product detail page with gallery, variants, price, and add-to-cart state.

## Debugging
Use `qa-debug-agent` to fix this hydration error and explain the root cause before changing code.

## Review
Run `/review-code` on the checkout flow and prioritize security and reliability issues.
