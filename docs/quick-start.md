# Quick Start

## Install into project
Copy `.cursor` to the root of your Next.js repo.

## Start with these agents
- next-architect
- ui-builder
- data-fetching-agent
- api-backend-agent
- db-agent
- qa-debug-agent

## Best first skills
- /plan-feature
- /build-feature
- /generate-api
- /create-db-schema
- /fix-bug
- /review-code

## Example flow
1. `/plan-feature admin orders dashboard`
2. Use `db-agent` if the feature needs schema changes.
3. Use `api-backend-agent` or `/generate-api` for backend work.
4. Use `ui-builder` or `/create-page` for the interface.
5. Run `/ship-checklist` before release.
