# Project Context

This repository is a modern full-stack product using:
- Next.js App Router
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL

Engineering preferences:
- server-first patterns
- modular architecture
- reusable UI
- predictable backend contracts
- maintainable code over clever code
- strong type safety
- scalable database design

Product expectations:
- fast iteration
- clean UX
- secure auth
- reliable data flows
- performance-conscious rendering
- production-ready implementation

When generating features:
- propose structure first
- identify data model needs
- define API/server-action contract
- implement UI with states
- explain tradeoffs briefly
